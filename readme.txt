== Description ==

Adds a system information page that include all the details on your WordPress configuration. That include server details, php configuration, WordPress configuration, current theme details, plugins details and hooks details. That allows for an easy copy/paste of the information you usually need to communicate to plugin authors for support.

== Installation ==

1. Unpack the *.zip file and extract the /system-information/ folder and the files.
2. Using an FTP program, upload the full /system-information/ folder to your WordPress plugins directory (Example: /wp-content/plugins).
3. Go to Plugins > Installed and activate the plugin.
4. Go to Tools > System Information.
